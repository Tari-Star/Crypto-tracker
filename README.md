# Crypto-tracker

This project is a crypto currency tracker that estimates the amount of crypto you can buy with whatever amount in USD/GBP/EUR invested.

# Technologies

1.html

2.Bulma/css

3.js

4.bootstrap

5.Api used - coinlore.net/free.currconv.com
